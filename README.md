# StudyHub
StudyHub adalah Sistem Manajemen Tugas Mahasiswa berbasis web (React.js &amp; Node.js) dengan antarmuka Kanban. Sistem ini memiliki CRUD tugas, unggah file, pengingat tenggat waktu, dan pertemanan untuk kolaborasi. Dilengkapi Study Music Widget, aplikasi ini dirancang dengan antarmuka bersih agar mahasiswa nugas lebih fokus dan terorganisir.
